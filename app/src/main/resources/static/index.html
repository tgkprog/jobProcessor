Project Overview

A robust, enterprise-ready Job Engine system designed to execute isolated processing logic bundled in JAR files. It handles the lifecycle of a job from submission to completion, including resource allocation and error tracking.

Built with **Spring Boot** and **Quartz Scheduler**, it provides a flexible architecture for long-running or CPU-intensive tasks.

  • **Hot Loading**: Attach JARs at runtime without restart
  • **Intelligent Timeouts**: Auto-terminates runaway jobs
  • **Real-time Scale**: Change thread pool size on-the-fly
  • **Detailed Auditing**: Full history with error codes and reasons
